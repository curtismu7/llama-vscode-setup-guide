# Set Up llama-vscode — The Easy Guide 🦙

A beginner-friendly, illustrated guide to running a **local AI coding assistant** in
VS Code with [llama.cpp](https://github.com/ggml-org/llama.cpp) and the
[llama-vscode](https://github.com/ggml-org/llama.vscode) extension — private, free,
and running entirely on your own computer.

### 👉 Read it on the web: https://curtismu7.github.io/llama-vscode-setup-guide/

Written in plain language so anyone can follow it. Covers:

- Installing the extension + `llama.cpp`
- Downloading a model (the "brain")
- The 5 model slots — completion / chat / embeddings / **tools** / **agent**
- "Vibe coding" with the agent (the part that edits files and runs commands)
- Troubleshooting — port collisions, keyboard shortcuts, stuck downloads
- A one-page cheat sheet

Built for Mac. The entire guide is a single self-contained `index.html` — no build
step, no dependencies. Open it in a browser and use **File → Print → Save as PDF**
to share it as a PDF.
