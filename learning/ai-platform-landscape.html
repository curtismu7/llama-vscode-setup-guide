AWS

🟠 Amazon Web Services
Build with the broadest AI/ML portfolio on the planet
Amazon BedrockFoundation Model API

Managed serverless API for 30+ foundation models (Anthropic Claude, Meta Llama, Mistral, Titan, etc.). No GPU management; pay-per-token. Supports RAG via Knowledge Bases, multi-step agents via Bedrock Agents, and fine-tuning.

Amazon SageMakerML Platform

End-to-end ML lifecycle: data prep, training, deployment, monitoring. Supports custom models, distributed training, real-time and batch inference. JumpStart library for pre-trained models.

Amazon QEnterprise AI Assistant

Generative AI assistant for developers (Q Developer, integrated in VS Code/JetBrains) and business users (Q Business for document Q&A). Fine-tunable on company data.

Amazon RekognitionVision AI

Image/video analysis: object detection, facial recognition, content moderation, text in image. REST API, no ML expertise required.

Amazon ComprehendNLP

Entity recognition, sentiment analysis, topic modeling, key phrase extraction. Supports custom entity/classification models trained on your data.

Amazon LexConversational AI

Build chatbots and voice bots with NLU. Integrates with Lambda for fulfilment. Powers Alexa.

Amazon KendraEnterprise Search

Intelligent search backed by NLP. Connects to S3, SharePoint, Salesforce. Bedrock Knowledge Bases is the newer evolution for RAG workloads.

Amazon Polly / Transcribe / TranslateSpeech & Language

Polly: text-to-speech. Transcribe: speech-to-text with custom vocabulary. Translate: 75+ language pairs with neural machine translation.

Key note: AWS Bedrock Agents support multi-step agentic workflows with tool use; integrate with Step Functions for orchestration. IAM-native auth — no separate identity layer needed for AWS-internal agents.

Microsoft

🔷 Microsoft Azure AI
AI everywhere, from infrastructure to copilots
Azure OpenAI ServiceFoundation Model API

Microsoft-hosted GPT-5.5, DALL-E, Whisper. Same models as OpenAI.com with enterprise SLAs, data residency, private networking (VNet), and Azure AD auth. Fine-tuning available.

Azure AI FoundryAI Development Platform

Unified portal (formerly Azure AI Studio) for model catalog, prompt flow, evaluations, and agent deployment. Model catalog includes Llama, Mistral, Cohere, Phi, and more.

Microsoft Copilot StudioAgent Builder

Low-code/no-code agent builder (formerly Power Virtual Agents). Build custom copilots with GPT-4 backbone, publish to Teams, web, Phone. Supports autonomous triggers and MCP.

Azure AI SearchRAG Backbone

Vector + keyword hybrid search with integrated chunking, embedding, and reranking. Core component of RAG pipelines on Azure. Semantic ranker powered by Microsoft's Bing ranking models.

Azure Machine LearningML Platform

End-to-end ML: experiments, pipelines, model registry, deployment to managed endpoints. MLflow integration.

Azure AI Content SafetySafety & Moderation

API for detecting harmful content (hate, violence, sexual, self-harm) in text and images. Used to gate LLM inputs/outputs in production.

Microsoft Semantic KernelAgent SDK

Open-source SDK (C#, Python, Java) for building LLM-powered apps and agents. Plugin/tool model, memory connectors, planner. Microsoft's answer to LangChain.

Phi-3 / Phi-4 (SLMs)Small Language Models

Microsoft's open-weight small models (3.8B–14B params). Strong reasoning relative to size; deployable on-device or edge. Available via HuggingFace and Azure AI Foundry.

Google

🟢 Google Cloud AI
From research breakthroughs to production AI
Vertex AIAI Platform

Google Cloud's unified ML/AI platform: train, deploy, and manage models. Model Garden hosts 160+ models (Gemini, Llama, Mistral, Claude via Anthropic partnership). AutoML for no-code training.

Gemini API / Google AI StudioFoundation Model API

Direct API access to Gemini 1.5 Pro/Flash/Nano and Gemini 2.x. Google AI Studio for rapid prototyping. 1M+ token context window. Multimodal: text, image, audio, video.

Vertex AI Agent BuilderAgent Framework

Build grounded agents with RAG (data store connectors), multi-turn conversation, and Google Search grounding. Replaces CCAI and Dialogflow CX for new projects.

Gemini for Google WorkspaceEnterprise Copilot

AI embedded in Gmail, Docs, Sheets, Meet. NotebookLM for document-grounded Q&A. Summarise, draft, and analyse directly in productivity tools.

Google Cloud Natural Language APINLP

Syntax analysis, entity recognition, sentiment, content classification. Pre-trained REST API with Healthcare NL and multilingual support.

Vision AI / Document AI / Video AIMultimodal AI

Vision: image labeling, OCR, face detection. Document AI: extract structured data from PDFs/forms. Video AI: transcription, content moderation, object tracking.

BigQuery MLML in Data Warehouse

Run ML models (linear regression, classification, ARIMA forecasting, even LLMs) directly in SQL on BigQuery data. No data movement necessary.

Cloud TPUs / Hyperdisk MLAI Infrastructure

Custom AI accelerators (TPU v5e/v5p); Hyperdisk ML for fast model loading. Foundation for training large models at Google scale.

IBM

🔵 IBM watsonx
Enterprise AI built for trust, transparency, and governance
watsonx.aiAI Studio

Train, validate, tune, and deploy foundation models and ML models. Model library includes IBM Granite (open-source), Llama, Mistral. Prompt Lab for prompt engineering.

IBM Granite ModelsFoundation Models

IBM's own open-source model family (Granite-13B, Granite-3B, Code models). Apache 2.0 licensed with full transparency on training data (curated, documented lineage). Designed for enterprise: smaller, cost-efficient, auditable.

watsonx.dataData Lakehouse

Open lakehouse architecture (Presto, Apache Spark, Apache Iceberg). Connects AI to governed data at scale. Reduces data warehouse costs by up to 50%.

watsonx.governanceAI Governance ⭐

Automated model risk management: bias detection, drift monitoring, explainability, regulatory compliance (EU AI Act, SR 11-7). This is IBM's key differentiator vs. other vendors.

Watson AssistantConversational AI

Enterprise chatbot builder. Integrates with watsonx.ai for LLM backbone. Deploys to web, phone, Slack, Salesforce.

Watson DiscoveryEnterprise Search

AI-powered document search and NLP enrichment. RAG-ready with watsonx.ai integration.

IBM OpenScale / OpenPagesRisk & Compliance

OpenScale: AI fairness, bias, and explainability monitoring. OpenPages: integrated risk management platform with AI for regulated industries.

IBM differentiator: IBM's focus is enterprise governance and transparency — auditability, bias monitoring, and compliance for regulated industries (banking, healthcare, insurance). Granite models publish full data lineage.

Anthropic

🟣 Anthropic
AI safety focused — powerful, steerable, and honest
Claude 5 — Opus 4.8 / Sonnet 5 / Haiku 4.5Foundation Models

Claude is Anthropic's frontier model family. Claude 5 family (2026): Opus 4.8 is most capable for complex reasoning; Sonnet 5 is the best price/performance balance; Haiku 4.5 is fastest/cheapest. The frontier line also ships Claude Fable 5 and Claude Mythos 5. 200K token context window. Claude 4.x and Claude 3.5/Opus 3 are prior generations.

Claude API (api.anthropic.com)Direct API

REST API with Messages endpoint. Tool use (function calling), vision (image inputs), streaming. Claude.ai for direct consumer access.

Claude on Amazon BedrockHosted API via AWS

Claude 5 family available via AWS Bedrock. Enterprise customers access Claude with AWS IAM, VPC, and data residency controls.

Claude on Google Vertex AIHosted API via GCP

Claude 5 family available via Vertex AI Model Garden. GCP-native auth, compliance, and private networking.

Constitutional AI (CAI)Safety Technique

Anthropic's training methodology: models are trained to follow a 'constitution' of principles (harmlessness, helpfulness, honesty). RLHF guided by AI feedback rather than purely human labels — more scalable.

Model Card & Eval TransparencySafety & Research

Published evals for jailbreaks, CBRN risk, autonomy, deception. Responsible Scaling Policy: capability thresholds trigger additional safety work before deployment.

Important note: Anthropic does not offer a full cloud AI platform — they are a model company. Their competitive advantage is frontier capability with safety-first training. Integrations via AWS Bedrock and Google Vertex AI cover enterprise deployment needs.

OpenAI

🟩 OpenAI
From research lab to the AI infrastructure of the internet
GPT-5.5 / GPT-5Foundation Models

GPT-5.5 is OpenAI's current flagship — multimodal (text/image/audio) with built-in reasoning modes, replacing the separate o-series. GPT-4o and o1/o3 are now previous-generation models.

OpenAI API — Chat CompletionsDirect API

Chat Completions, Files, Embeddings, Fine-tuning, Moderation, TTS, Whisper (STT), DALL-E (image gen). Batch API for async processing at 50% discount.

Assistants APIAgent Platform

Threads, Messages, Runs, Tool calls (Code Interpreter, File Search, Function Calling). Stateful: OpenAI manages conversation history and context automatically.

OpenAI Realtime APIVoice AI

Low-latency audio-in / audio-out via WebSocket. Powers voice agents with sub-300ms response time. Supports function calling mid-conversation.

DALL-E 3 / SoraGenerative Media

DALL-E 3: text-to-image, available via API and ChatGPT. Sora: text-to-video (limited access); sets state-of-the-art for video generation quality.

ChatGPT Enterprise / TeamEnterprise SaaS

Data not used for training, SSO/SCIM, higher rate limits, custom GPTs, GPT-5.5 access for all employees.

OpenAI Embeddings (text-embedding-3-large)RAG Backbone

State-of-the-art text embeddings for semantic search. 3072 dimensions; supports Matryoshka compression to 256 dimensions with minimal quality loss.

Fine-tuningCustomisation

Fine-tune GPT-5.5, GPT-5 mini, and legacy GPT-4o on custom datasets. Supports function calling and vision fine-tuning.

Ecosystem note: ChatGPT has 200M+ weekly active users. The OpenAI platform is the most widely adopted developer AI API. Microsoft Azure OpenAI Service mirrors the API with enterprise controls and data residency.

How to choose

AWSExisting AWS shops; broadest model choice + deep AWS service integration.
MicrosoftOffice 365 / Teams shops that want enterprise copilots with Azure compliance.
GoogleVery long context windows, multimodal (text/image/audio/video natively), or BigQuery ML.
IBMBanking, insurance, or healthcare needing auditable AI governance + regulatory compliance.
AnthropicSafety-critical applications, or when Claude's instruction-following quality is essential.
OpenAILargest developer ecosystem, most mature Assistants API, or real-time voice AI.

⭐ = significant differentiator. Table reflects public information as of mid-2026. Check each vendor's docs for current pricing and feature availability.
