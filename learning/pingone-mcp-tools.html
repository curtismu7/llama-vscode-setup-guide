⚠ What Changed Since 2026-06-11

No functionally new tools were added, but 25 tools were renamed and 1 tool was removed. All platform tools dropped the __platform_ snake_case naming for camelCase, and all DaVinci describe* tools became get*.

Platform — Environments

ToolDescription
listEnvironmentsLists PingOne environments accessible to the authenticated user; discover by name or status.
getEnvironmentReturns an environment's full configuration.
updateEnvironmentUpdates an environment by full replacement (get first — omitted fields are cleared). Region cannot change.
getEnvironmentServicesReturns the services (Bill of Materials) enabled for an environment.
updateEnvironmentServicesUpdates enabled services by full replacement — omitted services are removed.

Platform — Populations

ToolDescription
listPopulationsLists populations in an environment; discover by name.
getPopulationReturns a population's full configuration.
createPopulationCreates a new population in an environment.
updatePopulationUpdates a population by full replacement — name, description, password policy, theme references.

Platform — Users

ToolDescription
listUsersLists users; find by username, email, population, or status.
getUserReturns the full profile of a single user.
createUserCreates a new user, with or without an initial password.
updateUserUpdates profile attributes; only supplied fields change. Cannot change MFA or password.
manageUserAccountLocks or unlocks a user account; unlocking also resets the MFA lockout counter.
manageUserMfaReads or changes MFA settings: enable/disable, status, temporary bypass. The only tool that can change MFA enablement.
manageUserPasswordReads or changes password state: set, reset, force expiry, unlock, verify, OTP recovery. The only tool that operates on passwords.

Applications — Core

ToolDescription
listApplicationsLists applications; discover by name, type, or protocol.
getApplicationReturns the full configuration of a single application.
createApplicationCreates an application; subtypes: oidc_web_app, oidc_native, oidc_single_page, oidc_worker, oidc_device_authorization, saml. SAML supports raw xmlMetadata import.
updateApplicationUpdates an application; read-merge-write internally, pass only changed fields. Type and protocol cannot change.
deleteApplicationPermanently deletes an application; refuses system-managed apps. Supports dryRun.
getApplicationMetadataRetrieves the custom metadata stored on an application.
updateApplicationMetadataUpdates custom metadata; pass only changed keys, null removes a key.
getApplicationSecretRetrieves the masked client secret and rotation state; full value never returned.
regenerateApplicationSecretRotates the client secret; optional grace period keeps the old one valid.
deleteApplicationPreviousSecretRevokes the prior secret kept alive after rotation, ending its grace period early.

Applications — Attribute Mappings

ToolDescription

createApplicationAttributeMappingAdds a custom attribute to tokens/assertions from a user attribute or expression.
updateApplicationAttributeMappingChanges a mapping's value or required flag; the attribute name cannot change.
deleteApplicationAttributeMappingRemoves a custom mapping (built-in mappings cannot be removed).

Applications — Grants & Roles

ToolDescription
listApplicationGrantsLists resource grants — the resources and scopes an application may request.
createApplicationGrantGrants an application access to a resource's scopes; refused for system applications.
updateApplicationGrantChanges the scopes on an existing grant; the scope set is replaced as a whole.
deleteApplicationGrantRevokes an application's access to a resource by removing a grant.
listApplicationRoleAssignmentsLists admin roles a worker application can act with, and the scope of each.
createApplicationRoleAssignmentGrants a worker application an admin role at a chosen scope.
deleteApplicationRoleAssignmentRevokes an admin role from a worker application.
